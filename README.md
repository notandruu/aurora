# Aurora

Landing page for Aurora — an agentic AI assistant Chrome extension. Helps users browse smarter, automate repetitive tasks, and surface information contextually without leaving the browser.

## Features

- **Hero section** — animated Aurora orb with glassmorphic browser mockup demo
- **Feature cards** — AI productivity, intelligent search, context understanding, smart automation
- **Live demos** — AI chat interface, productivity view, search interface, shopping assistant
- **Pricing section** — tiered plan cards with glassmorphic styling
- **Testimonials** — community feedback carousel
- **FAQ** — animated accordion
- **Resizable navigation** — adaptive nav bar with smooth scroll
- **Fluid animations** — Framer Motion throughout, optimized scroll-triggered sections

## Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 15 (App Router) |
| UI | React 19 + shadcn/ui + Radix UI |
| Styling | Tailwind CSS v3 |
| Language | TypeScript |
| Animations | Framer Motion |
| Fonts | Geist Sans + Geist Mono |

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000).
